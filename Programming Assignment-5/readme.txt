Design and Analysis of Algorithm Programming Assignment-5
---------------------------------------------------------------------------------
Implementation of various Algorithms to solve Knapsack Problem


Notes:
----------------------------------------------------------------------------------
Done in C++

File-Info
----------------------------------------------------------------------------------
1) createkn01.cpp
2) backtrack.cpp
3) dynamic.cpp
4) bruteforce.cpp
5) Makefile
6) readme.txt

How to compile and run
----------------------------------------------------------------------------------
In remote.cs.binghamton.edu
1) make
2) For creation of knapsack:
	./createkn01 -k knapsack01.txt

3) For executing bruteforce: (creates a file output1)
	./bruteforce -k knapsack01.txt 

4) For executing dynamic programming: (creates two files output2 and entries2)
	./dynpro -k knapsack01.txt

5) For executing backtrack: (creates two files output3 and entries3)
	./backtrack -k knapsack01.txt
